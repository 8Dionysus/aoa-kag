# Release Patterns

This part owns the active KAG route for release, installation, and
post-release pattern candidates.

## Owns

- first-release, installation, and post-release pattern route docs;
- focused schema/example contract tests for release-pattern candidates;
- the owner-adoption stop-line for release-derived KAG candidates.

## Does Not Own

Release execution, owner adoption, proof certification, runtime state, or
office-service contracts owned by `parts/office-service-patterns/`.
