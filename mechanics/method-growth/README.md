# Method Growth

This package maps the common AoA Method-growth mechanic into the KAG layer.

## Mechanic card

Status: mapped common-center mechanic; active part-local routes exist for
pattern lineage, promotion dossiers, owner downlinks, and retirement.

### Trigger

Use this package when KAG pattern candidates, promotion dossiers, downlinks, or
retirement decisions need method-growth routing.

### Local owns

`aoa-kag` owns derived pattern lineage surfaces, promotion evidence bundle
shape, owner downlink posture, and retirement/pruning contracts at the
substrate layer.

### Stronger owner split

`Agents-of-Abyss` owns common method-growth law. `aoa-playbooks` owns recurring
method choreography. `aoa-techniques` owns technique canon. `aoa-skills` owns
executable skills. Owner repositories own local adoption.

### Inputs

Pattern candidates, promotion evidence bundles, owner signals, ToS dossier
bridges, downlinks, retirement decisions, and source refs.

### Outputs

KAG pattern candidate lineage, promotion dossiers, owner downlinks, retirement
signals, and next-owner routes.

### Must not claim

Method canon, technique canon, skill execution, owner adoption, proof verdict,
or automatic promotion from recurrence alone.
