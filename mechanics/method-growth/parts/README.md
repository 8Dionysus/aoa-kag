# Method-Growth Parts

`mechanics/method-growth/parts/` holds active KAG-side pattern-growth
sub-mechanics.

Use a part when a payload family has a local owner split, contract packet,
focused validation, and artifacts clearer inside the part than in root KAG
districts.

## Active parts

| Part | Role |
| --- | --- |
| `pattern-candidate-lineage` | Federation-harvest pattern candidates, lineage links, and derived pattern nodes. |
| `promotion-dossier` | Evidence bundles and ToS dossier bridge packets that stay below direct promotion. |
| `owner-downlink` | Owner consent, adoption dossier, downlink, and owner signal contracts. |
| `retirement` | Pattern retirement decisions and retirement downlink contracts. |
