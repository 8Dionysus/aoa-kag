# Sophian Threshold Packets

This part owns candidate-only KAG packet shapes for possible Sophian threshold
review.

## Role

It turns accepted KAG promotion pressure into reviewable threshold packet
candidates without granting KAG any Tree of Sophia canon authority.

## Inputs

- `config/sophian-threshold-packet-registry.source.json`
- part-local schemas for `agon-sophian-kag-packet`
- part-local examples for threshold packet payload shape
- owner-routed evidence and explicit boundary checks

## Outputs

- `generated/agon_sophian_kag_packet_registry.min.json`
- focused validation output from this part's validator and test

## Stop-lines

This part does not write ToS canon, create branches, run arena sessions, mutate
rank or trust, accept proof verdicts, or bypass authorial review.
