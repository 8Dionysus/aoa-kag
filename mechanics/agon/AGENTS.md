# AGENTS.md

## Applies to

This card applies to `mechanics/agon/` and all descendants.

## Role

`mechanics/agon/` is the KAG-local route for Agon pressure that asks whether
candidate patterns, promotion packets, or Sophian threshold packets can enter
the derived substrate without becoming verdict, rank, scar, canon, or source
truth.

## Read before editing

Read root `AGENTS.md`, `DESIGN.md`, `mechanics/AGENTS.md`,
`mechanics/README.md`, this package `README.md`, `PARTS.md`, and `PROVENANCE.md`.
Then read the current owner surface named by the changed route.

## Boundaries

- `Agents-of-Abyss` owns Agon law.
- `aoa-kag` owns only KAG-side derived candidate shape and validation posture.
- `aoa-evals` owns proof verdicts.
- `aoa-memo` owns memory evidence.
- `Tree-of-Sophia` owns Sophian canon and authored meaning.
- Only the parts listed in `PARTS.md` and `mechanics/topology.json` are active.
  Additional part directories need a part-local contract and validator first.

## Validation

Run `python scripts/validate_mechanics_skeleton.py`.
If Agon KAG registries move, also run the part-local builders, validators, and
tests before broader release checks.

## Closeout

Name the Agon KAG surface changed, whether a part was created, which stronger
owner stayed stronger, checks run, skipped checks, and the next owner route.
