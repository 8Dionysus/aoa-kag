# Agon Legacy

Agon legacy records former root path and legacy naming accounting for Agon KAG
surfaces that now route through active part-local homes.

It is not active topology, not a new-work entrypoint, and not a place to store
discarded material. Use the active part first, then use this directory only to
answer where an old path or old name went.

## Layout

- `INDEX.md`: former root paths and names mapped to active routes.
- `DISTILLATION_LOG.md`: dated accounting for the active-route distillation.
- `AGENTS.md`: local route law for legacy edits.
