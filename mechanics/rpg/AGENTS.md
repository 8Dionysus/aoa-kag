# AGENTS.md

## Applies to

This card applies to `mechanics/rpg/` and all descendants.

## Role

`mechanics/rpg/` is the KAG-local route for adjunct progression or readable
quest vocabulary when it helps interpret KAG obligations without creating a
hidden ontology.

## Read before editing

Read root `AGENTS.md`, `DESIGN.md`, `mechanics/AGENTS.md`,
`mechanics/README.md`, this package `README.md`, `PARTS.md`, and
`PROVENANCE.md`. Then read the quest, route, or public-facing KAG surface being
changed.

## Boundaries

- RPG language is adjunct vocabulary only.
- KAG does not create hidden ontology, runtime ledger, ranks, classes, or role
  rights from this package.
- Questbook and source owners keep stronger truth.
- No part directory is active until a part-local RPG contract and validator exist.

## Validation

Run `python scripts/validate_mechanics_skeleton.py`.
If quest or public vocabulary surfaces move, run focused KAG validators/tests.

## Closeout

Name RPG-adjacent vocabulary changed, stronger source preserved, checks run,
skipped checks, and next owner route.
