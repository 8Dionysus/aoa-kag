# RPG

This package maps the common AoA RPG mechanic into the KAG layer.

## Mechanic card

Status: mapped common-center mechanic; no active part directories yet.

### Trigger

Use this package when KAG quests or public route surfaces need adjunct
progression, campaign, or readable obligation vocabulary.

### Local owns

`aoa-kag` may own KAG-local wording that makes derived obligations readable
while keeping canonical ids, source refs, and owner routes authoritative.

### Stronger owner split

`Agents-of-Abyss` owns common RPG mechanic law. Questbook owns obligation
mechanics. `aoa-playbooks` owns campaign choreography. Runtime owners own live
progression state. Source repositories own meaning.

### Inputs

Quest records, KAG public route docs, consumer guide wording, and future
vocabulary overlays if they become necessary.

### Outputs

Adjunct readable vocabulary and route hints that remain weaker than canonical
KAG ids and source refs.

### Must not claim

Hidden ontology, runtime progression, rank, role rights, source meaning,
quest closure, or decorative game skin as authority.
