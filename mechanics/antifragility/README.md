# Antifragility

This package maps the common AoA Antifragility mechanic into the KAG layer.

## Mechanic card

Status: mapped common-center mechanic; active part-local routes exist for
projection health, projection quarantine, and retrieval-outage regrounding.

### Trigger

Use this package when KAG must handle retrieval outage, projection drift,
quarantine, regrounding chaos, or source-first recovery pressure.

### Local owns

`aoa-kag` owns bounded projection-health receipts, regrounding tickets,
quarantine posture, and KAG-specific stress docs that keep consumers returning
to stronger sources.

### Stronger owner split

`Agents-of-Abyss` owns common antifragility doctrine. `aoa-evals` owns proof.
`aoa-playbooks` owns recurring stress choreography. Source repositories own
repair and source meaning.

### Inputs

Projection-health examples, regrounding tickets, KAG stress docs, quarantine
docs, generated packs, and owner evidence refs.

### Outputs

Source-first recovery signals, projection quarantine posture, stress examples,
and owner-routed regrounding.

### Must not claim

One-score health, proof verdict, deletion authority, source repair, runtime
recovery, or hidden KAG cleanup authority.
