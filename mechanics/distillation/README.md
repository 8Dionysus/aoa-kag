# Distillation

This package maps the common AoA Distillation mechanic into the KAG layer.

## Mechanic card

Status: mapped common-center mechanic; no active part directories yet.

### Trigger

Use this package when source material is lifted into KAG packs, chunk maps,
route packs, or other derived projections.

### Local owns

`aoa-kag` owns source-linked lift manifests, generated derived payloads,
bounded normalization posture, and validation that keeps provenance visible.

### Stronger owner split

Source repositories own authored meaning. `Agents-of-Abyss` owns common
distillation law. `Tree-of-Sophia` owns ToS canon. `aoa-techniques` owns
technique canon. `aoa-evals` owns proof.

### Inputs

Source refs, KAG manifests, technique lift inputs, ToS chunk and route inputs,
schemas, examples, and generated output contracts.

### Outputs

Derived lift packs, chunk maps, route packs, generated projections, and
consumer-return refs.

### Must not claim

Summary as proof, ToS canon, technique canon, source replacement, memory truth,
or runtime activation.
