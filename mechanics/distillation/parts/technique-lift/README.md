# Technique Lift

This part owns the KAG-local operation route for lifting source-owned
`aoa-techniques` technique sections, metadata, relations, and provenance
handles into the current technique lift pack.

## Owner Route

- source owner: `aoa-techniques`
- local part: `mechanics/distillation/parts/technique-lift/`
- part source controls: `mechanics/distillation/parts/technique-lift/manifests/technique_lift_pack.json`
- part generated outputs: `mechanics/distillation/parts/technique-lift/generated/technique_lift_pack*.json`

## Stop-Line

This part may validate a bounded lift pack. It must not replace technique
canon, infer proof verdicts, or widen the active AOA-K surface set.
