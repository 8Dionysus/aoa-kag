# KAG Quest Lane

This lane holds `AOA-KAG-Q-*.yaml` source quest records for `aoa-kag`.

Use this lane for KAG-local obligations around derived substrate posture,
source-owned export dependencies, bridge and recurrence follow-through, and
bounded projection pilots.

Do not use it for authored source meaning, proof verdicts, memo truth, routing
authority, role law, playbook scenario composition, or runtime graph state.

## Current Split

- `done/`: closed provenance records that stay out of the public active index.
- `reanchor/`: open obligations whose route needs a stronger current anchor.
- `captured/`: open obligations preserved for later route shaping.

The lifecycle state in each YAML record must match its directory.
