# aoa-kag skill home

This directory is the canonical owner home for admitted `aoa-kag` procedures.
The current bundle is one global front door over the read-only five-tool KAG
MCP. Retrieve, relate, and research are its internal semantic modes; discover,
search, read, traverse, and explain remain tool operations.

The skill routes agents to owner-qualified sources and refuses to present
partial contracts or unavailable bindings as executable. It does not make KAG
the owner of proof, memory, routing, roles, playbooks, runtime, statistics,
techniques, or authored repository meaning.

Global exposure belongs to the OS user profile managed by `aoa-skills`. This
repository does not create a same-name `.agents/skills` projection.
